## Decisions Made (For ADRs)

- Next.js 16 over 15: Latest features, stable

- PostgreSQL on port 5435: Avoid conflicts with other local DBs
- pnpm over npm: Faster, disk efficient
- Vitest + Playwright: Unit + E2E coverage
- Tailwind v4 CSS config: Native CSS, no JS config needed
